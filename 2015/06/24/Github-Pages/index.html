简述

Github Pages 分 2 种，一是个人或组织的主页（xxx.github.io），一种是具体项目的主页（xxx.github.io/yyy）。均可以通过绑定域名，实现自定义域名。

注：其中 xxx 指代 github 用户名，yyy 指代 项目名。

  个人或组织主页 项目主页
仓库名 用户名.github.io 项目名
所在分支 master gh-pages
可拥有数量 <=1 无限制
默认访问 https://用户名.github.io https://用户名.github.io/项目名
自定义域名 可自定义 可自定义

个人或组织主页

  • 新建公开仓库，仓库名必须为 用户名.github.io；
  • 默认访问方式，https://用户名.github.io；
  • 自定义域名时，建立 CANAME文件并上传到仓库；

项目主页

  • 新建公开项目，仓库名任意（如：yyy）；
  • 新建 gh-pages 分支，主页文件必须放在此分支下；
  • 默认访问方式，https://用户名.github.io/项目名；
  • 自定义域名时，建立 CANAME文件并上传到 gh-pages 分支；
