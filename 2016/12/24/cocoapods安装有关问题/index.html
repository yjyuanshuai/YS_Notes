终端安装

1、卡在 Setting up CocoaPods master repo 后报错

经常会有卡在 Setting up CocoaPods master repo 很久后报以下错：

[!] /usr/bin/git clone https://github.com/CocoaPods/Specs.git master

Cloning into 'master'...
error: RPC failed: result=56, HTTP code = 200
fatal: the remote end hung up unexpectedly
fatal: early EOF
fatal: index-pack failed

可能的原因有 3 个：

  • gem 版本过低 
    sudo gem update --system
  • github 无法连接 
    ping github.com
  • .cocoapods目录下的配置信息错误 
    pod repo list

2、Ruby 源错

  http://taobao.ruby.org/       // 已失效
  https://taobao.ruby.org/      // 已不维护
  https://gems.ruby-china.org/  // 当前有效，2017-09-05

手动下载安装

1、pod: command not found

运行pod后显示：”pod: command not found”的错误。赋予/usr/local/bin给予执行与读取权限，执行以下命令。若不行执行sudo chmod 777 /usr/local/bin/。

sudo chmod 700 /usr/local/bin/

使用

1、pod search AFNetworking 时报以下错：

[!] Unable to find a pod with name, author, summary, or descriptionmatching `AFNetworking`

解决方法：

rm ~/Library/Caches/CocoaPods/search_index.json
