UIBezierPath 绘图

不与 CAShapeLayer 合用，纯 UIBezierPath 的使用一般是在 drawRect 方法中。

基本使用

  • 创建 UIBezierPath 对象；
  • moveToPoint设置起点；
  • 调用方法绘图（线、弧、圆、矩形）；
  • 设置属性（lineWidth / lineJoinStyle / lineCapStyle / color）；
  • stroke / fill 结束绘图。

属性

|属性|属性含义| |:–|:–| | lineWidth | 线宽 | | lineJoinStyle | 连接点样式 | | lineCapStyle | 终点样式 | | strock | 得到的是不被填充的 view | | fill | 内部填充的 view |

lineJoinStyle（连接点样式）

kCGLineJoinMiter  // 斜接
kCGLineJoinRound  // 圆滑
kCGLineJoinBevel  // 斜角

lineCapStyle（终点样式）

kCGLineCapButt    // 默认值，即不绘制端点
kCGLineCapRound   // 圆形端点
kCGLineCapSquare  // 方形端点

初始化方法

// 构造bezierPath对象，一般用于自定义路径。
+ bezierPath;

// 绘制圆弧路径，参数1是中心点位置，参数2是半径，参数3是开始的弧度值，参数4是结束的弧度值，参数5是是否顺时针(YES是顺时针方向，NO逆时针)。
+ bezierPathWithArcCenter: radius: startAngle: endAngle: clockwise:

// 根据某个路径绘制路径。
+ bezierPathWithCGPath:

// 根据某个CGRect绘制内切圆或椭圆（CGRect是正方形即为圆，为长方形则为椭圆）。
+ bezierPathWithOvalInRect:

// 根据某个CGRect绘制路径。
+ bezierPathWithRect:

/**
  * corners：指定哪个角为圆角
  *   UIRectCornerTopLeft
  *   UIRectCornerTopRight
  *   UIRectCornerBottomLeft
  *   UIRectCornerBottomRight
  *   UIRectCornerAllCorners
  *
  * cornerRadii：圆角的横纵半径
  */
+ bezierPathWithRoundedRect: byRoundingCorners: cornerRadii:

// 绘制每个角都是圆角的矩形，参数2是半径。
+ bezierPathWithRoundedRect: cornerRadius:

自定义路径常用

// 设置起点
- (void)moveToPoint:(CGPoint)point;   

// 绘制直线  
- (void)addLineToPoint:(CGPoint)point;  

// 绘制 3 次 贝塞尔曲线
- (void)addCurveToPoint:(CGPoint)endPoint controlPoint1:(CGPoint)controlPoint1 controlPoint2:(CGPoint)controlPoint2;

// 绘制 2 次 贝塞尔曲线
- (void)addQuadCurveToPoint:(CGPoint)endPoint controlPoint:(CGPoint)controlPoint;

// 绘制圆形曲线
- (void)addArcWithCenter:(CGPoint)center radius:(CGFloat)radius startAngle:(CGFloat)startAngle endAngle:(CGFloat)endAngle clockwise:(BOOL)clockwise

// 拼接曲线
- (void)appendPath:(UIBezierPath *)bezierPath;

应用

直线

UIColor *color = [UIColor redColor];
[color set]; //设置线条颜色

UIBezierPath* path = [UIBezierPath bezierPath];
path.lineWidth = 5.0;

path.lineCapStyle = kCGLineCapRound; //线条拐角
path.lineJoinStyle = kCGLineJoinRound; //终点处理

[path moveToPoint:CGPointMake(200.0, 50.0)];//起点

// Draw the lines
[path addLineToPoint:CGPointMake(300.0, 100.0)];
[path addLineToPoint:CGPointMake(260, 200)];
[path addLineToPoint:CGPointMake(100.0, 200)];
[path addLineToPoint:CGPointMake(100, 70.0)];
[path closePath]; //第五条线通过调用closePath方法得到的

// [path stroke];//Draws line 根据坐标点连线
[path fill];    //颜色填充

矩形

UIColor *color = [UIColor redColor];
[color set]; //设置线条颜色

UIBezierPath* path = [UIBezierPath bezierPathWithRect:CGRectMake(20, 20, 100, 80)];

path.lineWidth = 5.0;
path.lineCapStyle = kCGLineCapRound; //线条拐角
path.lineJoinStyle = kCGLineJoinRound; //终点处理

[path stroke];

圆形

UIColor *color = [UIColor redColor];
[color set];

UIBezierPath *path = [UIBezierPath bezierPathWithArcCenter:CGPointMake(100, 100) radius:90 startAngle:0 endAngle:TO_RADIAUS(120) clockwise:YES];

path.lineWidth = 5.0;
path.lineCapStyle = kCGLineCapRound;
path.lineJoinStyle = kCGLineJoinRound;
[path stroke];

圆弧

/**
  * center：原点
  * radius：半径
  * startAngle：开始角度
  * endAngle：结束角度
  * clockwise：是否顺时针方向
  */

+ (UIBezierPath *)bezierPathWithArcCenter:(CGPoint)center radius:(CGFloat)radius startAngle:(CGFloat)startAngle endAngle:(CGFloat)endAngle clockwise:(BOOL)clockwise

    [color set]; //设置线条颜色

制定矩形的某个角为圆角

UIColor *color = [UIColor redColor];
[color set];

UIBezierPath *path = [UIBezierPath bezierPathWithRoundedRect:CGRectMake(0, 0, 100, 100) byRoundingCorners:UIRectCornerTopRight cornerRadii:CGSizeMake(20, 20)];

path.lineCapStyle = kCGLineCapRound;
path.lineJoinStyle = kCGLineJoinRound;
path.lineWidth = 5.0;

[path stroke];
