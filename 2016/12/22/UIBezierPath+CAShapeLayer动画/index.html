CALayer

将路径显示的图案显示到视图上（3种方法）：

1、直接使用 UIBezierPath 的方法；

2、使用 CoreGraphics 绘制；

3、利用 CAShapeLayer 绘制。

UIBezierPath主要是用来绘制路径的，分为一阶、二阶…..n阶。一阶是直线，二阶以上才是曲线。而最终路径的显示还是得依靠CALayer。用CoreGraphics将路径绘制出来，最终也是绘制到CALayer上。


- (void)drawRect:(CGRect)rect
{
    UIColor *fillColor = [UIColor colorWithRed:0.0 green:0.722 blue:1.0 alpha:1.0];

    UIBezierPath *bezierPath = [UIBezierPath bezierPath];
    [bezierPath moveToPoint:CGPointMake(0, 0)];
    [bezierPath addLineToPoint:CGPointMake(rect.size.width - spaceWidth, 0)];
    [bezierPath addQuadCurveToPoint:CGPointMake(rect.size.width - spaceWidth, rect.size.height) controlPoint:CGPointMake(rect.size.width - spaceWidth + _deltaWith, rect.size.height * 0.5)];
    [bezierPath addLineToPoint:CGPointMake(0, rect.size.height)];
    [bezierPath addLineToPoint:CGPointMake(0, 0)];
    [bezierPath closePath];

    // 1、bezierPath方法
    //    [fillColor setFill];
    //    [bezierPath fill];

    // 2、使用CoreGraphics
    //    CGContextRef ctx = UIGraphicsGetCurrentContext();
    //    CGContextAddPath(ctx, bezierPath.CGPath);
    //    CGContextSetFillColorWithColor(ctx, fillColor.CGColor);
    //    CGContextFillPath(ctx);

    // 3.CAShaperLayer
    [self.layer.sublayers makeObjectsPerformSelector:@selector(removeFromSuperlayer)];
    CAShapeLayer *shapeLayer = [CAShapeLayer layer];
    shapeLayer.path = bezierPath.CGPath;
    shapeLayer.fillColor = fillColor.CGColor;
    [self.layer addSublayer:shapeLayer];
}

注：在用自定义的CAShapeLayer做动画时，建议在动画开始前先将动画属性与最终的属性值一致，再开始动画，不要使用removedOnCompletion控制最终的状态，这在WWDC苹果这么建议。

CAShapeLayer

|属性|属性含义| |:–|:–| | path | CGPathRef 对象，路径 | | lineWidth | 边线宽度 | | strokeColor | 边线颜色 | | lineDashPattern | 边线样式，默认是实线 | | lineDashPhase | 边线样式的起始位置 | | lineCap | 线终点的样式 | | lineJoin | 线拐点处的样式 | | strokeStart | 画边线的起点，即在路径上的百分比，0~1之间的CGFloat量 | | strokeEnd | 终点 | | fillColor | CGColorRef对象，图形填充色，默认为黑色 |

lineDashPattern

是一个 NSNumber 数组，数组中的值依次表示虚线中，单个线的长度，和空白的长度。

如：@[2,2,3,4]表示长度2的线，长度2的空白，长度3的线，长度4的空白，不断循环。

lineDashPhase

边线样式的起始位置，如上，就表示第 1 个长度2的线的起始位置。

lineCap


kCALineCapButt    // 不设置样式
kCALineCapRound   // 圆形
kCALineCapSquare  // 方形

lineJoin


kCALineJoinMiter  // 三角
kCALineJoinRound  // 圆形
kCALineJoinBevel  // 斜切
