iOS沙盒结构

沙盒目录获取

根目录

  NSHomeDirectory();

Documents

保存应用运行时生成需要持久化的数据，iTunes 同步时__会__备份。

如：游戏存档

- (NSString *)dirDoc{
      // [NSHomeDirectory() stringByAppendingPathComponent:@"Documents"];

      // NSDocumentDirectory: 表明我们正在查找 Documents 目录的路径
      // NSUserDomainMask: 表示将搜索限制在引用程序的沙盒内
      // YES: iOS中主目录的全写形式是/User/userName，这个参数填YES就表示全写，填NO就是写成‘‘~’’，我们一般填YES。
      NSArray *paths = NSSearchPathForDirectoriesInDomains(NSDocumentDirectory, NSUserDomainMask, YES);

      // 每个应用只有一份 Documents 目录，因此只需取数组首位
      NSString *documentsDirectory = [paths objectAtIndex:0];
      NSLog(@"app_home_doc: %@",documentsDirectory);
      return documentsDirectory;

      // stringByAppendingPathComponent + 文件名
}

Library

体积大但相对不重要的数据。

-(void)dirLib{

    //[NSHomeDirectory() stringByAppendingPathComponent:@"Library"];
    NSArray *paths = NSSearchPathForDirectoriesInDomains(NSLibraryDirectory, NSUserDomainMask, YES);
    NSString *libraryDirectory = [paths objectAtIndex:0];
    NSLog(@"app_home_lib: %@",libraryDirectory);
}

Cache

程序运行时需要持久化存储的数据，iTunes__不会__备份。

  -(void)dirCache{
    NSArray *cacPath = NSSearchPathForDirectoriesInDomains(NSCachesDirectory, NSUserDomainMask, YES);
    NSString *cachePath = [cacPath objectAtIndex:0];
    NSLog(@"app_home_lib_cache: %@",cachePath);
}

Preferences

偏好设置数据。iTunes 不会 备份

利用 NSUserDefaults 来记录设置。

Tmp

临时数据。不备份

-(void)dirTmp{
    //[NSHomeDirectory() stringByAppendingPathComponent:@"tmp"];

    NSString *tmpDirectory = NSTemporaryDirectory();
    NSLog(@"app_home_tmp: %@",tmpDirectory);
}
