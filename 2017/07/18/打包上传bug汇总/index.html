平时测试运行都没有问题，但 achive 时会报错。

1、Unsupported Architecture

1）描述：

  Unsupported Architecture. Your executable contains unsupported architecture '[x86_64, i386]

1) 情景：

生产环境打包时报错，而平时开发、测试打包并未发现为题。

2) 原因：

导入的第三方库，存在不支持的框架。

3) 解决：

工程中增加 Script 检查并删除第三方不支持的 framework等。

2、Invalid Bundle

1）描述：

  Invalid Bundle - A nested bundle contains simulator platform listed in CFBundleSupportedPlatforms Info.plist key

2) 情景：

平时开发没有任何问题，但在上传 App Connected 时，总是提示版本无效。苹果邮件说明如上 bug 描述。

3) 原因：

添加第三方库时，该库内存在的 Info.plist，且 Info.plist 中存在支持平台是模拟器。

4) 修改：

找出第三方库中所有 Info.plist 文件（注意：bundle 中的文件在XCode中搜不到），将 key 为 CFBundleSupportedPlatforms 的 value 换成 iPhoneOS。
